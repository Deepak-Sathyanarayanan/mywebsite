Overview

This project analyzes social-media text and classifies tweets into sentiment categories using a large, labeled Twitter dataset.

The workflow includes exploratory analysis, text preprocessing, tokenization, embedding generation using GloVe, and neural-network-based modeling. It demonstrates how noisy user-generated text can be transformed into structured sentiment predictions at scale.

Features

  • Explores tweet characteristics such as stopwords, word count, and special characters
  • Applies preprocessing including stemming, regex cleaning, and stopword removal
  • Uses visualization techniques such as word clouds to inspect sentiment patterns
  • Performs tokenization and GloVe-based word embedding
  • Feeds processed text into a neural network for sentiment prediction

Architecture

Raw Tweets → Text Cleaning / Regex / Stopword Removal → Tokenization → GloVe Embeddings → Neural Network → Sentiment Prediction

Dataset

The project uses the Sentiment140 dataset, which contains 1,600,000 tweets collected through the Twitter API.

The dataset is labeled with 0 for negative sentiment and 4 for positive sentiment, making it suitable for supervised sentiment classification on large-scale social text.
