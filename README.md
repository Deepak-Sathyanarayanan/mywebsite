# mywebsite

Personal website to showcase my projects in Data Science, Machine Learning, NLP, and AI.

## Overview

This repository contains my personal portfolio website, built to highlight my background, technical skills, and selected projects. It serves as a central place to present my work in data science and related areas, including applied machine learning, natural language processing, analytics, and end-to-end project delivery.

## Features

- Personal portfolio homepage
- About page with professional background
- Contact page
- Project gallery
- Individual project pages with visuals and summaries
- Resume access

The website currently showcases projects in areas such as:

- Retrieval-Augmented Generation (RAG)
- Speech Recognition
- Twitter Sentiment Analysis
- Fraud Detection

## Tech Stack

- CSS
- GitHub Pages
