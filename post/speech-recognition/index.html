Overview

This project focuses on recognizing spoken audio inputs by converting raw audio into frequency-based features and training a deep learning model to classify commands.

The workflow uses speech data provided by Google and extracts MFCC features before feeding them into a TensorFlow model. The final model was also exposed through FastAPI, making the project an end-to-end machine learning application rather than only a notebook experiment.

Features

  • Predicts spoken audio commands from one-second recordings
  • Uses the Google Speech Commands dataset as the training source
  • Converts audio into MFCC frequency representations for modeling
  • Applies TensorFlow-based deep learning for classification
  • Productionized with FastAPI for serving predictions

Architecture

Audio Files → MFCC Feature Extraction → TensorFlow Model → Predicted Speech Command

Dataset

The project uses Google's Speech Commands dataset, a public collection created to support voice-command recognition research.

According to the original project page, the dataset contains roughly 65,000 one-second utterances of 30 short words spoken by thousands of different people, making it well suited for practical speech classification experiments.
