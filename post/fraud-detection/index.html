Overview

This project is focused on detecting fraudulent claims submitted by healthcare providers using machine learning and structured claims data.

The workflow combines multiple input sources through joins and unions, explores the data to identify predictive patterns, and applies dimensionality reduction and model tuning to improve classification performance. The original project notes report 94% model accuracy using a Random Forest classifier.

Features

  • Predicts potentially fraudulent healthcare-provider claims
  • Combines multiple input sources through joins and unions
  • Uses exploratory data analysis to identify informative variables
  • Applies PCA and hyperparameter tuning to improve performance
  • Uses a Random Forest classifier for final prediction

Architecture

Claims Data Sources → Data Integration → EDA / Feature Selection → PCA → Random Forest Classifier → Fraud Prediction
